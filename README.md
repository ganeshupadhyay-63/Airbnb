# Airbnb Project

This is a clone of the Airbnb web application built with HTML, CSS, JavaScript, and backend technologies.

## Features
- User login and signup
- Listing properties
- Booking system
- Responsive design

## Tech Stack
- Frontend: HTML, TalwindCSS, JS, React
- Backend: Node.js, Express
- Database: MongoDB
